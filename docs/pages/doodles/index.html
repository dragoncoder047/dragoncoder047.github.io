Graphic Design Class (2025)

The first two here were made in Adobe Illustrator. I would have rather used Inkscape but alas, it was not installed on the lab computers, and students were not allowed to install new programs (even portable installations) on university devices.

This one was just freehanded in a few minutes as filler for the last page of a page-layout project. I used the comics above on the other pages.

This one was done using the same technique I now use - import one or more reference photographs, trace them, and then delete the reference photos. I think I used four reference images on this one.

These two were made in Adobe Photoshop. I would have rather used GIMP but it was also not installed.

I went to town with the “puppet warp” tool on this one. There was really no other way to do things since the restrictions on this project were that we had to use stock images only and could not use the pen or bucket fill tools.

For the one below, I could not find any stock images of a person sitting in the pose I was looking for, so I did the next best thing and took a photo of myself.

Interesting note: I do not own a Playstation Portal nor any other handheld gaming console (Steam Deck, Nintendo Switch, etc). In the original photo I used, I was holding a book. But thanks to the magic of image editing you’ll never notice that!
